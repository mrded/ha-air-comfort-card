# Air Comfort Card

A custom Home Assistant card that visualizes indoor air comfort using temperature and humidity sensors. It displays a circular "comfort dial" with a moving dot indicator showing whether conditions are cold, warm, dry, humid, or comfortable â€” plus 24-hour history charts for temperature, humidity, and CO2.

![Air Comfort Card](https://raw.githubusercontent.com/mrded/ha-air-comfort-card/main/screenshot.png)

## Features

- Circular comfort dial with color-coded zones and a moving indicator dot
- Temperature, humidity, and CO2 readings
- 24-hour history charts
- Configurable comfort thresholds
- Visual editor for easy setup
- Theme-aware design

For configuration details, examples, and development info, visit the [GitHub repository](https://github.com/mrded/ha-air-comfort-card).
